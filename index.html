Game Time

Les statistiques à portées de mains

Le concept

L'application Game Time récolte les statistiques des joueurs et les présente sous la forme de plusieurs slides. L'utilisateur peut ainsi consulter les statistiques générales et avancées mais aussi l'historique du joueur. Destiné aux journalistes mais aussi aux staffs d'équipes professionnels, ces derniers peuvent retrouver les données en direct mais aussi sauvegarder dans leurs favoris les joueurs qu'ils souhaitent suivre.

Le public concerné

Le concept est déstiné aux journalistes et commentateurs sportifs pour la préparation du match. Les entraineurs peuvent également utiliser l'application pour adapter leur stratégie.

La problématique

Les statistiques sont éparpillées sur plusieurs sites ou applications mais ces dernières se concentrent essentiellement sur les équipes et non les joueurs. Avec tout ces chiffres, il est difficile de pouvoir trouver uniquement les statistiques des joueurs

La solution

Nous voulons proposer une application mobile et un site internet regroupant toutes les données entourant un joueur de football. Historique, données générales et avancées, toutes ces statistiques sont regroupées sur la fiche du joueur

Le prototype

Game Time se présente sous la forme d'un application mobile avec comme exemple la dernière coupe du monde et notamment la demi-finale France-Belgique. La performance de Samuel Umtiti est alors illustrée par ses statistiques avancées pour permettre au staff techniques, aux analystes d'observer son impact sur le match.
